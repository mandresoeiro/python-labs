# Guia Completo do Repositório Python Labs

## 1. Objetivo
Este repositório serve como base para estudar Python através de pequenos projetos.

## 2. Como criar um novo projeto
```
cd scripts
./novo_projeto.sh 02 conversor_moedas
```

## 3. Como rodar um projeto
```
cd projetos/02_conversor_moedas
python3 main.py
```

## 4. Como documentar
Cada projeto possui o arquivo NOTAS.md para registrar o aprendizado.

## 5. Como publicar documentação
```
mkdocs serve
mkdocs build
```
